# 制式改造 (ColonistModification)

RimWorld 1.6 模组——为殖民者批量标准化植入仿生部件和异种基因。

## 解决的痛点

- **逐人逐部位手动操作**：原版要打开每个人物的健康面板，从长列表里找到对应植入物，左右部位再区分，几十个殖民者操作上百次
- **人工盯材料状态**：铁、零部件够不够全靠记忆或手动查仓库
- **无批量模板**：不能一次定义"给这几个人装仿生臂×2 + 仿生腿 + 仿生眼"，等着自动推进
- **做完一侧就算完成**：装完左臂系统不会提醒右臂还没装，直到你主动检查

## 功能

- **自定义改造模板**：创建手术模板，选择植入物（仿生臂/腿/眼/脊椎等）和基因植入（异种胚芽），一次定义批量复用
- **批量分配**：在概览页为多个殖民者分配同一模板
- **自动检测与通知**：系统定期检查资源、医生、身体部位，条件满足时弹出确认窗或自动添加手术。缺材料会显示具体物品名和库存数量
- **部位分立追踪**：左臂/右臂、左眼/右眼各自独立追踪完成状态，不分左右部位（脊椎、基因植入）单独处理
- **基因植入支持**：模板可指定目标异种，自动搜索地图上匹配的胚芽并绑定到手术单。植入完成后通过检测殖民者异种变化判定完成
- **实时完成判定**：直接从殖民者身体 hediff 和异种检测进度，不存盘不额外记录
- **手术并发控制**：限制同时进行的手术数量，防止医院爆满
- **左右取消不丢数据**：取消模板分配后重新分配，已完成部位记录保留

## 开发

```
dotnet build
```

输出：`publish/1.6/Assemblies/ColonistModification.dll`

推荐创建目录链接到 mod 目录：

```
mklink /D F:\SteamLibrary\steamapps\common\RimWorld\Mods\ColonistModification F:\RiderProjects\UnlimitedFoundations\ColonistModification\publish
```
